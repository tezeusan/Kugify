# Kugify

A FastAPI backend project.

## Installation

```bash
uv venv .venv && source .venv/bin/activate
uv pip install -r requirements-dev.txt
````

## Development

```bash
uvicorn app.main:app --reload
```

## Useful Commands

```bash
make format   # Format code with Black
make lint     # Lint code with Ruff
```
